## 音源運用ルール

## 保存先
- `01_最新楽曲_フル`: 最新で受領したフル楽曲（1曲分）
- `02_過去楽曲_フル`: 旧曲のフル楽曲アーカイブ
- `03_ショート用分割`: Aメロ/Bメロ/サビなどに分割した短尺書き出し

## 基本フロー
1. 会長がフル楽曲を `01_最新楽曲_フル` に保存
2. CEOが構成を確認し、`Aメロ/Bメロ/サビ/落ちサビ` を切り出し
3. 切り出し結果を `03_ショート用分割` に保存
4. 次の新曲受領時、前回のフル楽曲を `02_過去楽曲_フル` へ移動

## PR期間の切替ルール（固定）
- `01_最新楽曲_フル` に新曲が入った時点で、PR対象はその新曲へ即時切替する
- 切替と同時に、直前までの最新曲は `02_過去楽曲_フル` へ移動する
- `02_過去楽曲_フル` に移動した楽曲は「PR終了」とし、新規PR投稿の主対象にしない
- 例外的に再投稿する場合は、比較企画や振り返り企画に限定する

## 推奨命名
- フル: `YYYY-MM-DD_曲名_full.wav`
- 分割: `YYYY-MM-DD_曲名_Aメロ_15s_v1.wav`
- 分割: `YYYY-MM-DD_曲名_サビ_20s_v1.wav`
